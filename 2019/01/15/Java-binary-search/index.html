算法题：Java实现二分查找

二分查找算法的前提条件

  • 必须有序（单调递增或递减）：因为每次都有通过判断大小来确定查找范围
  • 存在明确上下界：因为每次查找都要把数组一分为二
  • 必须通过索引能访问：因为确定了上限下限，必须得通过中间值的索引去去中间值的大小

综上所述：有序数组是最适合使用 二分查找算法

因为二分查找算法每一次都会减少一半的数据(每一次都除以二) 所以二分查找的 时间复杂度 为 O(logN)

代码实现

/**
 * 二分查找
 *      前提: 有序
 *      时间复杂度: O(logN)
 * @author Alan Wang
 * @date 2018/12/5 21:59
 */
public class BinarySearch {
    public static void main(String[] args) {
        int[] arr = new int[1000000];

        for (int i = 0; i < arr.length; i++) {
            arr[i] = i+i;
        }
        System.out.println();
        System.out.println(search(arr, 18));
    }

    /**
     * 查询给定数组种是否存则某个值val
     * @param val
     * @return
     */
    static int search(int[] arr, int val) {
        int left = 0;
        int right = arr.length-1;

        int time = 0;
        while (left <= right) {
            time ++;
            // 取中间值
            int mid = (right+left)>>1;
            System.out.println(left+"--------第"+time+"次查找，中间值："+mid+"---------"+right);
            if (val < arr[mid]) {
                right = mid-1;
            } else if (val > arr[mid]) {
                left = mid+1;
            } else {
                return mid;
            }
        }
        return -1;
    }
}
